## Bienvenido a Open Wearables

Plataforma de código abierto que unifica datos de dispositivos wearables de múltiples proveedores y habilita insights de salud impulsados por IA mediante automatizaciones en lenguaje natural. Crea aplicaciones de salud más rápido con una sola API, widgets integrables y notificaciones inteligentes por webhook.

## Qué Hace

Open Wearables ofrece una API unificada y un portal para desarrolladores para conectar y sincronizar datos de múltiples dispositivos wearables y plataformas fitness. En lugar de implementar integraciones separadas para Garmin, Fitbit, Oura, Whoop, Strava y Apple Health, puedes usar una sola plataforma para acceder a datos de salud normalizados y crear insights inteligentes mediante automatizaciones impulsadas por IA.

**Visión**: las automatizaciones de Health Insights desbloquean el potencial de los datos de usuario y te permiten impulsar negocios de una forma completamente nueva. Al definir condiciones en lenguaje natural, los desarrolladores pueden crear notificaciones e insights inteligentes que respondan a patrones de salud, habilitando nuevos tipos de aplicaciones y servicios.

<Note>
  **Para uso personal**: esta plataforma también puede ser usada por personas que quieran interactuar con sus propios datos de wearables. Conecta tus dispositivos, usa el Asistente de Salud con IA para conversar con tus datos y configura automatizaciones de insights personales, todo manteniendo tus datos en tu propia infraestructura.
</Note>

## Por Qué Usar Open Wearables

**Para desarrolladores que crean apps de salud:**
- 🔌 Integra múltiples proveedores de wearables mediante una sola API, en lugar de mantener implementaciones separadas
- 📊 Accede a datos de salud normalizados entre distintos dispositivos, como frecuencia cardiaca, sueño, actividad y pasos
- 🤖 Crea insights de salud y automatizaciones impulsadas por IA usando lenguaje natural
- 🏠 Solución autohospedada, con despliegue en tu propia infraestructura y control total de los datos
- 🚀 Sin dependencias de terceros para la funcionalidad principal; ejecútalo localmente con `docker compose up`
- 🧩 Widgets integrables para sumar la plataforma fácilmente a tus aplicaciones

**El problema que resuelve:**

Crear una app de salud compatible con múltiples wearables normalmente requiere:
- ⏱️ Semanas de desarrollo por cada integración de proveedor
- 🔐 Gestionar distintos flujos OAuth y APIs para cada servicio
- 📦 Manejar múltiples formatos y unidades de datos
- 🔧 Mantener varios SDK y adaptarse a cambios en las APIs

Open Wearables absorbe esa complejidad para que puedas concentrarte en construir tu producto.

## Casos de Uso

- 👤 **Uso personal**: cualquier persona puede autohospedar la plataforma para conectar sus propios wearables, conversar con sus datos usando el Asistente de Salud con IA y configurar insights personales con privacidad y control total
- 🏃 **Apps de coaching fitness**: conecta wearables de usuarios para ofrecer recomendaciones de entrenamiento personalizadas. Los coaches pueden crear usuarios, compartir enlaces por WhatsApp y probar capacidades de insights con IA
- 🏥 **Plataformas de salud**: agrega datos de pacientes desde varios dispositivos y configura automatizaciones para alertas de salud
- 💪 **Aplicaciones de bienestar**: da seguimiento y analiza la actividad de tus usuarios a través de distintos wearables con insights impulsados por IA
- 🔬 **Proyectos de investigación**: recopila datos de salud estandarizados desde múltiples fuentes
- 🧪 **Pilotos de producto**: equipos no técnicos pueden probar la plataforma compartiendo enlaces de conexión con usuarios, sin necesidad de tener una app propia

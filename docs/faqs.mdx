## Preguntas Frecuentes

### Preguntas Generales

#### ¿Qué es Open Wearables?

Open Wearables es una plataforma de código abierto que ofrece una API unificada para acceder a datos de salud desde múltiples dispositivos wearables y plataformas fitness. Soporta proveedores como Garmin, Fitbit, Oura, Whoop, Strava y Apple Health.

#### ¿Open Wearables es gratis?

Sí. Open Wearables es open source y de uso gratuito. Puedes autohospedarlo en tu propia infraestructura.

#### ¿Puedo usar Open Wearables para uso personal?

Open Wearables está pensado principalmente para desarrolladores y entrenadores profesionales, pero también puede ser usado por personas que quieran interactuar con sus propios datos. Puedes conectar tus dispositivos, y pronto estarán disponibles el Asistente de Salud con IA y las automatizaciones de health insights.

### Preguntas Técnicas

#### ¿Cómo empiezo?

Sigue nuestra [guía Quickstart](/quickstart) para configurar Open Wearables en minutos.

#### ¿Qué proveedores wearable están soportados?

Actualmente, Open Wearables soporta:
- Suunto
- Garmin
- Polar

Se están agregando más proveedores de forma constante. Revisa las [guías de configuración de proveedores](/providers/supported) para ver la lista más reciente, o consulta el estado actual de solicitudes e implementaciones en [GitHub Issues](https://github.com/the-momentum/open-wearables/issues?q=is%3Aissue+label%3Aprovider-request) con la etiqueta `provider-request`.

#### ¿Puedo autohospedar Open Wearables?

Sí. Open Wearables está diseñado para autohospedarse. Puedes desplegarlo en tu propia infraestructura con control total de los datos.

#### ¿Cómo integro Open Wearables en mi aplicación?

Puedes integrarlo usando:
- La API REST unificada
- Widgets integrables para una integración más sencilla (próximamente)
- Notificaciones por webhook para actualizaciones en tiempo real (próximamente)

Revisa la [Referencia API](/api-reference/introduction) para ver guías detalladas de integración.
